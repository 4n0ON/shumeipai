树莓派（英语：Raspberry Pi），是一款基于Linux系统的只有一张信用卡大小的单板机计算机。它由英国的树莓派基金会所开发，目的是以低价硬件及自由软件刺激在学校的基本的电脑科学教育。[4][5][6][7][8]

树莓派被生产是通过有生产许可的两家公司：Element 14/Premier Farnell和RS Components。这两家公司都在网上出售树莓派。[9]

树莓派配备一枚700MHz博通出产的的ARM架构BCM2835处理器，256MB内存（B型已升级到512MB内存），使用SD卡当作储存媒体，且拥有一个Ethernet，两个USB接口，以及HDMI(支持声音输出)和RCA端子输出支援。Raspberry Pi只有一张信用卡大小，体积大概是一个火柴盒大小，可以执行像雷神之锤III竞技场的游戏和进行1080p影片的播放。操作系统采用开源的Linux系统，比如Debian、ArchLinux，自带的Iceweasel、KOffice等软件能够满足基本的网络浏览，文字处理以及计算机学习的需要，分A,B两种型号，其中售价分别是A型25美元，B型35美元。基金会从2012年2月29日开始接受B型的订货。[10]

树莓派基金会提供了基于ARM的Debian和Arch Linux的发行版供大众下载。还计划提供支持 Python 作为主要编程语言[11][12], 支持BBC BASIC[13] , (通过 RISC OS 映像或者Linux的"Brandy Basic"克隆)[14], C， 和 Perl等编程语言.

[编辑] 历史

2006年，早期的树莓派的概念来源于Atmel的ATmega644单片机。其原理图和PCB布局可供公众下载。基金会受托人埃·厄普顿（Eben Upton）汇集了一批教师学者和计算机爱好者，制作一套启发孩子的计算机[15]。这种计算机是受到了1981年的艾康计算机公司的BBC Micro计算机的启发。第一台ARM计算机的原型被装在与一个U盤的大小相同的一个“盒子”中。它一端有一个USB接口，而另一端有一个HDMI接口。

早期的树莓派alpha测试板在运行中. 它的布局板和beta和成品板不同.

[编辑] 发布之前

2011年8月，五十块Alpha测试版的电路板被制造。这些电路板的功能与原计划中的B型的相同，但体积更大的以容纳调试接口。该电路板的演示展示了它能运行LXDE桌面的Debian，以1080p的分辨率运行Quake 3，通过HDMI播放全高清MPEG-4影片。

2011年10月，一个RISC OS 5系统的实验版本能够正常运行，并且展示这个成果给大众，并且在经过一年多的开发后，在2012年11月发布给大众使用。[16][17][18][19]

2011年12月，在100个未组装的电路板中，有二十五个B型的Beta测试版电路板被组装和测试。测试版电路板的元件布局与生产电路板相同。在电路板的设计上发现一个错误，即CPU上的一些引脚没有保持高电平，这个错误在首次生产时已被修复。Beta版电路板已证明可以启动Linux，播放1080p的电影预告片和运行Rightware Samurai 的OpenGL ES基准测试。

2012年的第一周，最初的10个电路板在eBay上被拍卖。有人匿名购买，并捐赠给座落在英国萨福克的计算机历史博物馆。这十个电路板（总零售价220英镑）拍卖一共筹集了超过16000英镑的资金。预计在2012年2月结束发布之前，该基金会的服务器将会疲于应付浏览者一再刷新他们浏览器造成的负载。

[编辑] 发布

首批10000张电路板是在台湾和中国大陆制造，而不是在英国。这是部分由于各个组成部分要进口关税缴付，而成品不用。中国制造商还只需要4周的准备时间，但在英国需要12周。节省下的资金可再投资于该基金会的其他研究和开发活动。

2012年3月公布首批运送的延误，是不正确安装以太网端口的结果。但该基金会预计，如果需要的话，未来批次的制造数量可以毫无困难地增加。

“我们确保我们能得到大量（以太网接口与磁性元件），在协助提供部件方面，Premier Farnell和RS Components（两个经销商）是极好的。”厄普顿说。

最初的销售开始2012年2月29日06:00 UTC。同时，它宣布，A型原本有128 MB的RAM，但在发布之前要升级到256 MB。该基金会的网站也宣布“在项目的开始6年后，我们的第一次开发运行就要接近结束了 - 虽然树莓派的故事才是个开始。” 他们在英国本地的两个特许专卖店销售，由于过重的流量导致官方网站网站暂停服务，他们立即推出自己的网站。同时站长也恳求，“朋友～你能别如此频繁的按F5吗，将会使服务器瘫痪的。”虽然还未经证实，报告显示，有超过200万的意向书或预购。树莓派官方Twitter帐户报道，Premier Farnell在发售的最初几分钟内就销售一空，而在第一天的RS Components的预购订单超过10万。截至2012年9月，已售出大约500,000板。[20]

2012年3月，制造商被报道要接受“健康的数量”的预购订单。[21]

[编辑] 发布之后

2012年4月16日有报道第一个买主收到了他们的树莓派[22][23]。截至2012年5月22日，已经运送超过20000个树莓派[24] 。2012年7月16日宣布，每一天有4000个被制造，让树莓派进入了大批量生产阶段[25][26]。9月5日树莓派基金会公布了B型第二次修订版[27] 。

在2012年9月6日基金会宣布大批量的树莓派将在英国制造，由位于威尔士Pencoed市的索尼工厂生产。该基金会估计，该工厂每月将生产30,000个，并将创造大约30个新的就业机会[28][29]。在2012年10月，它的两个主要分销商的一些客户等待他们的订货竟超过了6个月。据报道，这是由于在采购CPU和保守的销售预测上的困难。[30]

[编辑] 基于树莓派的超级计算机

南安普敦大学的计算机工程师们使用64个树莓派构建了一个廉价“超级计算机”'[31]。 64块树莓派采用如乐高积木一般的方式累加而成，使用MPI（Message Passing Interface）标准通过高速以太网实现连通。

该“超级计算机”拥有64块处理器，具备1TB的存储空间（每块树莓派装备了一张16GB SD卡 ），总造价在2500英镑以下，命名为Iridis-Pi，取自该校的另外一台超级计算机Iridis（全欧洲Win Server运行最快的计算机）。

Simon Cox教授表示他们尝试了在该超级计算机上运行Pi测试，不过没有公布运算测试。

当然，该设备的主要目的是为了启发学生们进行高性能计算方面的研究，甚至在自家屋里也可以轻松构建！感兴趣的爱好者可以查看详细构建指南和对应源代码。(点击这里查看）

运行X window的树莓派系统.

最初销售的树莓派是B型，计划2013年早些时候推出A型。A型有一个USB接口，并没有以太网控制器，将耗资小于B型（它有两个USB端口和一个10/100以太网控制器）。

虽然A型没有一个RJ45以太网接口，但依然可以连接到网络（使用用户提供的USB以太网或Wi-Fi适配器）。现实中外部的以太网适配器A型和B型一个内建之间没有差异，因为B型的以太网端口实际上是一个内置的USB以太网适配器。由于是典型的现代计算机，树莓派当然支援USB键盘和鼠标。

树莓派没有硬件实时时钟，故一个操作系统必须使用网络时间服务器，或在开机时要求用户时间与日期才能获得时间、日期以及资料夹日期的相关讯息。不过，如果用附带时钟的备用电池（如DS1307的），则可以通过I2C接口来获得时间。

在2012年4月20日，树莓派基金会公布出树莓派的A型和B型的电路原理图。[32]

影片硬件加速技术H.264于2012年08月24号公怖并且在许可证明也提到编码。之前已预计编码将会于摄影模组同时发行。同时，树莓派发怖两个可分开购买的编码，MPEG-2以及微软的VC-1。树莓派也同样发怖过会支援CEC，将能经由电视控制器对其控制。

在2012年9月5日，修订版2.0的板子被公布，有一些较小的修正和改进。[33]

在2012年10月15日，树莓派基金会宣布所有新的树莓派的B型将配有512MB RAM芯片，而不是256MB。[34]

[编辑] 软件

API连接的方框图

树莓派基金会使用基于Linux内核的操作系统。在2012年7月，基金会发布了目前它推荐使用的操作系统，Raspbian，这是基于Debian的Linux发行版，并对于树莓派的硬件的做了系统优化。


[编辑] 操作系统(OS)

这是一个在树莓派上运行的操作系统(operating system, OS)的清单, 包括已经移植好的或者正在被移植的过程中的：

  • 完整OS:
    • AROS
    • Linux
      • Android
        • Android 4.0 (Ice Cream Sandwich)[46]
      • Arch Linux ARM
      • Debian Squeeze
      • Firefox OS
      • Gentoo Linux[47]
      • Google Chrome OS
      • Raspberry Pi Fedora Remix
      • Raspbian[48] (Debian Wheezy port with faster floating point support)
      • Slackware ARM (formerly ARMedslack)
      • QtonPi a cross-platform application framework based Linux distribution based on the Qt framework
      • WebOS
        • Open webOS[49]
    • Plan 9 from Bell Labs[50]
    • RISC OS
    • Unix
      • FreeBSD[51]
      • NetBSD[52][53]
  • 多功能轻型发行版:
    • Moebius, a light ARM HF distribution based on Debian. It uses Raspbian repository, but it fits in a 1 GB SD card. It has just minimal services and its memory usage is optimized to keep a small footprint.
    • Squeezed Arm Puppy, a version of Puppy Linux (Puppi) for the ARMv6 (sap6) specifically for the Raspberry Pi.[54]
  • 单功能轻型发行版:
    • Instant WebKiosk a browser-only operating system
    • IPFire
    • OpenELEC
    • Raspbmc
    • XBMC
    • XBian

[编辑] 附件

在2012年5月，该基金会在其博客中汇报了他们的测试原型相机模块。该样机采用的是1400万像素的模块，其正式发布的版本可能会使用较低的分辨率，以减少资源负荷。而该版本很可能在2013年的上半年发布。[55]

许多树莓派附件器件和外壳都可以从第三方供应商得到。[56]这些包括的树莓派基金会批准，专为教育目的而设计的Gertboard板，这板扩展了树莓派的GPIO引脚，允许控制连接到LEDs, 开关, 模拟信号, 传感器和其他器件. 它还包括一个可选的 Arduino 兼容控制器连接到Pi.[57]

[编辑] 社区

自由及开放源代码软件公司AdaCore的Jamie Ayre描述树莓派项目中最令人兴奋的部分是社区。社区博客罗素·戴维斯（Russell Davis）说，社区力量使基金会专注于文档和教学。社区爱好者发展的围绕着平台的杂志，例如电子杂志《The MagPi》[58]。在英国的圣母高中[59][60]的信息和通信技术教师 艾伦·O'Donohoe[61][62][60]的带领下，社区展开了名为“树莓酱”(英语：Raspberry Jam)的一系列社区教学活动，在英国各地举行[61], 和其他更远的地方[63]。并且，以教师为主导的社区开始建立一个众包（CrowdSource）的工作计划。[64]

爱好者经常聚会，交流使用心得，称这类聚会为“树莓酱”(Raspberry　Jam)。英国曼彻斯特、马汉莱斯、美国硅谷、新加坡等地也举办过树莓酱聚会。

[编辑] 评价

正面评价：

技术作家格林·穆迪（Glyn Moody）于2011年5月描述了树莓派是一个“潜在的 BBC Micro​ ​2.0”项目，不是要取代而是要补充PC兼容机。[65] 在2012年3月在ITPRO上，斯蒂芬·普里查德(Stephen Pritchard)呼应了“BBC Micro的接班人”的情绪。[66]

在英国的计算机历史博物馆大力支持树莓派项目，感觉是它可以“迎来一个新的时代”[67]。在发布之前，树莓派板子被展示给在剑桥的ARM首席执行官 Warren East ，概述了谷歌的想法，以提高英国科学和技术教育。[68] 次世代报告(Next Gen report)的合著者，Alex Hope，认为儿童接触计算机编程的兴奋是充满希望。[69]

在2012年10月树莓派赢得了T3的“年度创新奖”。[70]


负面评价：

然而，Harry Fairhead表明更多的重点应放在提高现有硬件上提供的教育软件， 例如，利用类似于谷歌手机编程软件工具(Google App Inventor)让编写程序返回学校，而不是增加新的硬件选择[71]。 Simon Rockman，写在一个ZDnet博客认为，青少年将"有​​更好的事情要做"，不用管在20世纪80年代发生的事。[72]

一些开源项目批评了缺乏可用的硬件文档（树莓派网站上的常见问题，博通（Broadcom）公司没有为了BCM2835发布一个完整的data sheet），这可用的硬件文档发布本应该将使得其他操作系统移植到树莓派变得更容易。[73][74]

[编辑] 应用

截至2012年1月，在英国已收到来自公立学校的和独立（私立）学校的董事会的查询，后者数量是前者数量的5倍左右。希望企业将赞助购买条件较差的学校[15]。分销商Premier Farnell集团的CEO说，在中东的一个国家政府已表示有兴趣提供了每一个女学生一个电路板，以提高她们的就业前景。[75][76]
