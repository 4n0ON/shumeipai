1.安装usb摄像头，http://www.eeboard.com/bbs/thread-1510-1-1.html
2.安装强大的网络监控软件motion，输入
sudo apt-get install motion （如果安装不了的，可以先sudo apt-get update 一下）
3.配置motion，编辑motion.conf文件
sudo vi /etc/motion/motion.conf （或者运行sudo xedit /etc/motion/motion.conf）

4.找到”control_localhost on “和”webcam_localhost on“这两行，改为以下两行后，保存退出
control_localhost off
webcam_localhost off
5.运行motion软件，输入
motion -n
6.在pc上用IE浏览器打开，192.168.99.53是我的树莓派ip，换成你们自己的
配置网页：http://192.168.99.53:8080
