1、必须安装python开发版，即python-dev，不然后面很多软件都会安装不成功，提示目录无法找到之类的。（教训）

查看源代码
打印帮助
1sudo apt-get install python-dev

正常情况下，以上在线安装命令即可安装成功，但运气不好的是，笔者在安装时，在线命令安装资源包404。
python2.7-dev_2.7.3~rc2-2.1_armhf.deb找不到，后来只能手动找了以下包安装。

查看源代码
打印帮助
1wget http://ftp.debian.org/debian/pool/main/p/python2.7/python2.7-dev_2.7.3-6_armhf.deb
2sudo dpkg -i python2.7-dev_2.7.3-6_armhf.deb

2、安装python的GPIO模块
Google Code项目地址：http://raspberry-gpio-python.googlecode.com

查看源代码
打印帮助
1wget http://raspberry-gpio-python.googlecode.com/files/RPi.GPIO-0.4.1a.tar.gz
2tar -xvzf RPi.GPIO-0.4.1a.tar.gz
3cd RPi.GPIO-0.4.1a
4sudo python setup.py install

3、安装webopio
Google Code项目地址：http://code.google.com/p/webiopi/

查看源代码
打印帮助
1wget http://webiopi.googlecode.com/files/WebIOPi-0.5.3.tar.gz
2tar xvzf WebIOPi-0.5.3.tar.gz
3cd WebIOPi-0.5.3
4sudo ./setup.sh

4、首先设置或修改webopio的密码

查看源代码
打印帮助
1sudo webiopi-passwd

用户名是：webiopi
如不修改密码，默认密码是：raspberry

5、启动webopio

查看源代码
打印帮助
1sudo python -m webiopi 8000

端口可自行修改，默认是8000

6、让webopio后台运行，否则按ctarl+c就会断掉

查看源代码
打印帮助
1sudo /etc/init.d/webiopi start
2and
3sudo /etc/init.d/webiopi stop

7、设置webopio随系统启动

查看源代码
打印帮助
1sudo update-rc.d webiopi defaults

8、用浏览器打开IP地址访问管理界面

http://192.168.1.106:8000/webiopi/


控制方法：
a.点击外侧的“IN/OUT”切换GPIO的工作模式，输入/输出
b.输出模式下，点击内侧的数字，切换输出电平，高/低
c.输入模式下，内测数字代表GPIO的输入状态
