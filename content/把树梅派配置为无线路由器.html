首先需要一个受驱动支持的无线网卡，配置的结果就是把树梅派的网络连接通过无线网卡分享出来。做分享的无线网卡当然就不能同时来连接网络了。
安装hostapd
  1. sudo apt-get install hostapd
复制代码
配置hostapd.conf
  1. interface=wlan0
  2. driver=nl80211
  3. ssid=网络名称
  4. channel=6
  5. hw_mode=g
  6. wpa=2
  7. wpa_passphrase=密码
  8. wpa_key_mgmt=WPA-PSK
  9. wpa_pairwise=TKIP
  10. rsn_pairwise=CCMP
复制代码
安装isc-dhcp-server
  1. sudo apt-get install isc-dhcp-server
复制代码
配置/etc/dhcp/dhcpd.conf，在末尾添加
  1. subnet 192.168.111.0 netmask 255.255.255.0 {
  2.         range 192.168.111.50 192.168.111.150;
  3.         option routers 192.168.111.1;
  4.         option domain-name-servers 8.8.8.8;
  5.         option ip-forwarding off;
  6.         option broadcast-address 192.168.111.255;
  7. }
复制代码
启动hostapd
  1. sudo hostapd hostapd.conf -B
复制代码
设置无线网卡地址
  1. ifconfig wlan0 192.168.111.1
复制代码
启动isc-dhcp-server
  1. /etc/init.d/isc-dhcp-server start
复制代码
配置iptables
  1. echo 1 > /proc/sys/net/ipv4/ip_forward
  2. iptables -t nat -A POSTROUTING -o eth0 -j MASQUERADE
复制代码
这样就可以把树梅派的的有线网络连接通过无线网卡信号分享出去，可以用手机或平板测试一下。如果对iptables足够了解还可以配置很多高级内容。树梅派仅仅配置成为一个无线路由可能还不够，可以配合文件服务器一起用。并不是所有网卡都能配置成功，下面是简单测试结果：
