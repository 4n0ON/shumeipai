BT下载transmission-daemon
这一段感谢@诺亚
安装transmission-daemon
sudo apt-get install transmission-daemon
默认情况下，无法使用其他电脑访问transmission的web下载界面。为了便于远程下载文件，修改一下配置
关闭服务
sudo service transmission-daemon stop
修改配置文件
/etc/transmission-daemon/settings.json
"rpc-authentication-required": true,.  关闭用户验证，改成false
"rpc-whitelist-enabled": true,.  关闭白名单过滤，改成false
重新启动服务
sudo service transmission-daemon start

用其他电脑访问
http://树莓派的IP:9091
