由于很多Raspberry Pi新手都会遇到不少操作问题，DesignSpark为各位Raspberry Pi用户制作了四个Raspberry Pi的国语版视频教程! 视频教程包括Raspberry Pi 中文设置环境安装示范、智能拼音输入法安装示范和音频与USB快闪磁盘安装示范。若您有其它问题也可以在DesignSpark中发问。


Raspberry Pi 中文设置环境安装示范(上) (国语)

Raspberry Pi 中文设置环境安装示范(下) (国语)
http://www.designspark.com/zh-hans/knowledge/chi/raspberry-pi-language-setting-setup-B

Raspberry Pi 智能拼音输入法安装示范(国语)
http://www.designspark.com/zh-hans/knowledge/raspberry-pi-pinyin-setup

Raspberry Pi 音频与USB快闪磁盘安装示范(国语)
