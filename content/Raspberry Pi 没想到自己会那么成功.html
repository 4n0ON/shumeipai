若讨论起今年开源硬件中的风云设备，必然要提 Raspberry Pi，价格最低仅 25 美元，大小与一张信用卡相仿的它，如今成为了喜好自己动手的 Geek 们的宠物。

CNN 的记者 Nick Glass 采访了 Raspberry Pi 的发明者，剑桥大学的教授 Eben Upton，谈及发明 Raspberry Pi 的最初愿望，消费人群，未来前景等等。

愿望：让小孩子编程

此前 Upton 曾接收 BusinessInsider 的采访，他谈到 Raspberry Pi 的目标是为了让儿童也能学会编程而开发的。而面对着 Glass，他更详细地解释了这一目标的原因。

作为一名剑桥大学的教授，每年他和以及同事们都努力去吸引学生来学习计算机科学，然而现实很骨感，每年报修计算机科学的学生的数量越来越少。上世纪 90 年代，报名学习计算机科学的学生达 500 名，如今则少于 250 名。

学生对计算机科学的兴趣下降，原因是什么呢？与电脑的普及有关，当人们自然而然地接受了某一事物的存在，就会缺乏探索的兴趣。“电脑？那不是玩《魔兽争霸》用的么？”

因此他希望通过 Raspberry Pi，能够让学生重新感受到上个世纪 80 年代的气氛——当时电脑没有那么先进，但能够引发好奇心。

Raspberry Pi 最有竞争力的地方是价格

Upton 当初没想到 Raspberry Pi 会如此受欢迎——一年前，他与团队对这部机器的期望是，“一生的时间内卖出去一万部。”当 Raspberry Pi 开始销售的时候，消费者的热情很快把他们给淹没，网站因为承受不住突然爆发的流量而宕机。今年，也就是 Raspberry Pi 开售的第一年，这部超低价电脑的销量高达百万。

当问及 Raspberry Pi 最具竞争力的地方在哪里时，Upton 的回答是“价格”。从一开始，他们就希望这部电脑的价格与一本教科书的价格差不多（在国外，教科书的价格很贵），这样一个小孩也能负担得起。

Raspberry Pi 最初的主要消费人群不是儿童

Raspberry Pi 的最初设计目标是希望小孩子也能玩编程。现在是否有学校采购 Raspberry Pi，并用于课堂教学中呢？

截止今年 9 月，Raspberry Pi 的主要买家是那些和 Upton 相似的人，受过良好教育，对科技感兴趣的人，他们用 Raspberry Pi 来开发自己的项目。是的，基于 Raspberry Pi 的硬件和软件，我们已经听过不少，有为它移植操作系统的，也有人用它来操控机器人的等等。

为了达成最初的愿望，9 月份之后，Upton 和团队开始转变方向，将 Raspberry Pi 单独卖给儿童，以及学校。不过 Upton 并未透露有儿童买家的数量是多少，有多少学校采购，采购数量是多少。

今年最酷的 Raspberry Pi 创新是什么？

围绕 Raspberry Pi 的创新有许多，比如基于它的声控咖啡机、VOIP 电话、可戴式的医学设备，还有将 Raspberry Pi 变为一部笔记本电脑的等等。

最有趣当属 Zack Freedman 的项目，他动手改装 Raspberry Pi，把它变成类似 Google Glass 那样酷炫的东西：

他将 Myvu Crystal 视频眼镜拆了一个下来作为 Raspberry Pi 的显示器，然后又在手腕上绑了一个带有鼠标触摸板的无线键盘，给电脑安装了 Ouillix pIO 专门开发的 microSD 卡卡槽，还加装了一块硬盘，总体成本大约 100 美元。
