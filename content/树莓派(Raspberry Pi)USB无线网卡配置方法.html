首先查看树莓派外设兼容列表：http://elinux.org/RPi_VerifiedPeripherals#USB_Wi-Fi_Adapters，以确定自己的无线网卡是否可用，一般免驱的大多可用。这种USB无线网卡，一般只要二三十块钱就可以搞定，推荐易迅购买。

另外新版的镜像，包含的驱动比较多。

1、查看自己的无线网卡是否生效
查看挂载的USB设备

查看源代码
打印帮助
1sudo lsusb

我的RPI显示如下：

查看源代码
打印帮助
1pi@raspberrypi:/$ sudo lsus b
2Bus 001 Device 001: ID 1d6b:0002 Linux Foundation 2.0 root hub
3Bus 001 Device 002: ID 0424:9512 Standard Microsystems Corp.
4Bus 001 Device 003: ID 0424:ec00 Standard Microsystems Corp.
5Bus 001 Device 004: ID 0bda:8176 Realtek Semiconductor Corp. RTL8188CUS 802.11n WLAN Adapter

最后一行标有802.11n WLAN Adapter即是我的无线网卡，表明设备可用。

2、测试wifi信号

查看源代码
打印帮助
1sudo iwlist wlan0 scan

细心看，找到自己无线路由器的ssid(即用手机搜wifi，显示的wifi名字，路由器中可以配置)
笔者ssid是：TP-LINK_FB5906

3、编辑网卡配置信息

查看源代码
打印帮助
1sudo nano /etc/network/interfaces

将wlan0的部份修改为：

查看源代码
打印帮助
1auto lo
2
3iface lo inet loopback
4iface eth0 inet dhcp
5
6allow-hotplug wlan0
7iface wlan0 inet static
8    wpa-ssid 你要连接的wifi ssid
9    wpa-psk 你的wpa连接密码
10address 192.168.1.106   # 设定的静态IP地址
11netmask 255.255.255.0   # 网络掩码
12gateway 192.168.1.1     # 网关
13network 192.168.1.1     # 网络地址
14#wpa-roam /etc/wpa_supplicant/wpa_supplicant.conf
15iface default inet dhcp

笔者把IP设在了静态的了，你总不想每次SSH登陆还要换IP吧！建议设成静态的
