Raspberry Pi，是一款基于Linux系统的个人电脑，配备一枚700MHz的处理器，256Mb内存，支持SD卡和Ethernet，拥有两个USB接口，以及 HDMI和RCA输出支持。有消息称，虽然Raspberry Pi看起来非常的迷你——只有一张信用卡大小，但是它能够运行像《雷神之锤三：竞技场》这样的游戏和进行1080p视频的播放。和最终售价 200 美元的 OLPC 不一样，体积大概是一个火柴盒大小，硬件基础是 ARM，操作系统采用开源的Linux系统，比如 Debian、ArchLinux，自带的Iceweasel、KOffice 等软件能够满足基本的网络浏览，文字处理以及计算机学习的需要，分A,B两中型号，其中A型售价仅25美元。

发布

首批10000张电路板是在台湾和中国大陆被制造，而不是在英国。这是部分由于各个组成部分要进口关税缴付，而成品不用。中国制造商还只需要四个星期的准备时间，但在在英国需要12周。可节省资金再投资于该基金会的其他研究和开发活动。

2012年3月公布首批运送的延误，是不正确安装的以太网端口的结果。但该基金会预计，未来批次的制造数量可以毫无困难地增加，如果需要的话。

“我们确保我们能得到大量（以太网接口与磁性元件），派睿电子和RS Components（二级经销商）是完美的帮助原始码元件。”厄普顿说。

最初的销售开始2012年2月29日06:00 UTC。同时，它宣布，A型，原本有128 MB的RAM，在发布之前要升级到256 MB。该基金会的网站也宣布“项目​​以来的6年后，我们几乎在我们发展的第一次运行结束，虽然它只是Raspberry Pi传奇的开始。”他们在英国本地的两个特许专卖店销售，由于过重的流量导致官方网站网站暂停服务，他们立即推出自己的网站。在一个点上的站长恳求，“伙计~你能不要如此频繁的按F5，你将会使服务器瘫痪。”虽然还未经证实，报告显示，有超过200万的意向书或预购。Raspberry Pi官方Twitter帐户报道，派睿电子首次推出几分钟内出售一空，而参加了第一天的RS元件前的订单超过10万。

2012年3月，制造商要采取预购的“健康号”。

2012年4月16日报道第一个买主收到了他们的Raspberry Pi。

截至2012年5月22日，已经发运超过20000台Raspberry Pi。

硬件

最初销售的是B型，计划晚些时候推出A型。A型有一个USB接口，并没有以太网控制器，将耗资小于B型（它有两个USB端口和一个10/100以太网控制器）。

虽然A型没有一个RJ45以太网接口，但依然可以连接到网络（使用用户提供的USB以太网或Wi-Fi适配器）。现实中外部的以太网适配器A型和B型一个内建之间没有差异，因为B型的以太网端口实际上是一个内置的USB以太网适配器。由于是典型的现代计算机，通用USB键盘和鼠标是与Raspberry Pi兼容。

Raspberry Pi没有硬件时钟，故一个操作系统必须使用网络时间服务器，或在开机时要求用户获得文件的时间和日期冲压时间和日期时间信息。不过，如果用带实时时钟的备用电池（如DS1307的），则可以通过I2C接口来获得时间。 2012年4月20日放出Raspberry Pi的型号A和型号B由基础示意图。

硬件加速的视频（H.264）编码尚未公布，树莓PI在原则上GPU虽然有处理这样的任务的能力，但Raspberry Pi的价格只包括一个视频解码不编码的许可费。最终责任方的许可条款的条件是，MPEG LA。如果公布的摄像头模块被释放的情况将改变Raspberry Pi，其价格将包括对H.264编码的牌照费。这是是没有GPU代码的支持，为实施许可证的免费编码器Ogg Theora格式。

    订购和开箱就省略了，相信看到这里的人大部分都已经拿到了Raspi，值得一提的是，收到货后可以提交自己的地址，在网上可以看到Raspi的全球分布情况，网址是：http://www.rastrack.co.uk/

    安装系统可以参考官方的下载页，http://www.raspberrypi.org/downloads，有Raspbian “wheezy”、Arch Linux ARM、QtonPi三个系统共下载，用ImageWriter工具写入到SD卡里就行了。

    但是，有一种更加简便，更加快捷的方式安装系统，那就是Berryboot,安装方法见官方网站，http://www.berryterminal.com/doku.php/berryboot，值得注意的是，由于一张SD卡安装多个操作系统，文件系统有Berryboot接管，Berryboot会根据启动的操作系统的不同，自动挂载文件系统，所以，debian在进行raspi-config的时候不要选择“expand_rootfs”，因为Berryboot已经为你做好了。当然，如果选择了也问题不大，会在启动的时候多出一条错误信息而已。

    安装好raspbian后第一次启动会有一个raspi-config，这个界面也可以后期在终端中通过sudo raspi-config调出。

    安装完成后，可以设置一下raspbian：

更新系统： sudo apt-get update

安装中文字库： sudo apt-get install ttf-wqy-zenhei

输入法:sudo apt-get install scim-pinyin

安装chrome:sudo apt-get install chromium

安装gedit:sudo apt-get install gedit

耳机口输出声音 sudo amixer cset numid=3 1（0:自动，1：耳机，2：HDMI） 

声音测试 sudo aplay /usr/share/sounds/alsa/Front_Center.wav

    安装完字体后再把Local设置成zh_CN.UTF-8，否则会出现乱码，因为没有字库支持。

    想要看视频最好是安装BerryBoot里边的OpenELEC，也就是为Raspi定制XBMC，这个版本比较完美，我使用几天下来感觉一些功能胜于目前更新频繁的XBinn。
